Selectors In HTML

Elements Selector

The elements selector selects HTML elements based on the element name

Id selector

The id selector uses the id attribute of an HTML elements to select the a special element.The id of an element is unique within pages,so the id Selector is send to select one unique element!

Class Selector

The class selector selects HTML with specific class attributes .To selects elements with a specific class.write a period() character ,followed by class name .
